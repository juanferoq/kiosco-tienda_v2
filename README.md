# Kiosco Asistente Virtual

## Instrucciones para Railway

1. Sube esta carpeta a GitHub
2. Conecta el repositorio en railway.app
3. En Railway → Variables, agrega las 5 variables de .env.example
4. Railway despliega automáticamente y te da la URL

## Variables de entorno requeridas
- CLAUDE_API_KEY
- ELEVENLABS_API_KEY
- ELEVENLABS_VOICE_ID
- HEYGEN_API_KEY
- HEYGEN_AVATAR_ID
